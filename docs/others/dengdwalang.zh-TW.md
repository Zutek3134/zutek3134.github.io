# Deng Dwa Lang
一一二年三月廿六

---

<div style="writing-mode: vertical-rl; text-orientation: upright;">
<br>所剩無幾的十七歲，
<br>轉瞬即逝，弦斷絲竹殘；
<br>韶光荏苒，年輪又一圈；
<br>昔日信念，未嘗實現，趑趄止渡邊。
<br>
<br>稚氣未泯的十八歲，
<br>音韻悠揚，空山詩詞響；
<br>白駒過隙，達達猶叵及；
<br>今朝經驗，銳氣似箭，扁舟駛向前。
<br>
<br>夜暮宵分時，宵柝響萬家；
<br>黎明朝暾際，朝暘揭霧霞。
<br>搖曳弄枝椏，掠影乃風颳；
<br>十七變十八，犯罪用刑法。
</div>
