    3. 我 Discord 帳號編碼正好為「#3134」（非常幸運）。
    4. 自十歲始便做遊戲。
    5. 會國文、英文、閩南語、日文、法文，惟前兩者是真正精通。

    9. 經常瀏覽教育部《重編國語辭典》消遣。
    10. 非常自豪以 GitHub 作為私人維基。
